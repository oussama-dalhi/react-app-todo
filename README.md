# React Todo App

A simple React Todo/Habit Tracker app built while learning modern React and TypeScript.

## What I Practiced

- React components
- Props and state
- TypeScript basics
- Context API
- Component structure
- Tailwind CSS styling
- Vite project setup


## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

```bash
npm install
```
